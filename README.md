# 🎥 AI YouTube Video Analyzer

An intelligent **Agentic AI-powered YouTube Video Analyzer** built using **Agno**, **Groq**, and **Streamlit**. This project extracts insights from YouTube videos using advanced AI workflows, enabling users to quickly understand video content, generate summaries, and interact with video information efficiently.

---

## 🚀 Features

* 🔍 Analyze YouTube videos using AI agents
* 🧠 Agentic AI workflow powered by Agno
* ⚡ Ultra-fast inference with Groq LLMs
* 📄 Automatic video summarization
* 📝 Extract key insights and important points
* 💬 Interactive Streamlit-based user interface
* 🎯 User-friendly and responsive design
* 🔗 Simple YouTube URL input

## 🔑 Environment Variables

Create a `.env` file in the root directory and add your API keys:

```env
GROQ_API_KEY=your_groq_api_key
```

---

## 🧠 How It Works

1. User enters a YouTube video URL.
2. The application extracts video information and transcript data.
3. Agno-powered AI agents process the content.
4. Groq LLM generates summaries and insights.
5. Results are displayed interactively using Streamlit.
